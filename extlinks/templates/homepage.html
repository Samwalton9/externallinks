Wikilink

The Wikilink tool collates and presents data on the addition, removal, and total number of links to websites on Wikimedia projects. It tracks the page-links-change event stream for link additions and removals for specified URL Patterns, and queries the externallinks table to retrieve totals.

The tool currently only supports The Wikipedia Library program, however support for other programs and partnerships is planned.
